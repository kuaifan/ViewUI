## 说明 

运行如下命令安装相关依赖包并启动项目，如果您只需要构建出皮肤的CSS样式文件，可以直接调到使用说明

```
// 安装依赖包
yarn install

// 启动工程
yarn dev

// 发布
yarn publish

// 使用 docker 发布
docker run -itd -v $(pwd):/view --name node-view node:11.15.0
docker exec -it -w /view node-view /bin/bash
yarn publish

```
